## Browser Extensions

Venice.ai has been tested and is known to work with the following browsers:

#### [Brave Leo AI (Bring Your Own Model)](https://brave.com)

Requires Version 1.76.52 or greater. Configure Venice in the Leo settings under "Bring your own Model":

* Label: Venice.ai

* Model request name: `default` (Or enter any Venice API model ID)

* Server Endpoint: [https://api.venice.ai/api/v1/chat/completions](https://api.venice.ai/api/v1/chat/completions)

* Context size: 4000 (Can be increased, but will expend more tokens)

* API Key: Paste your API key from the \[API dashboard]\(https://venice.ai/settings/api

* System Prompt: Customize to your desire
