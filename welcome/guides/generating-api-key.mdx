---
title: Generating an API Key
---

Venice's API is protected via API keys. To generate a key, you must be a [Venice Pro User](https://venice.ai/pricing).

Once you've upgraded, within your user settings on Venice, you should see this option:

<Frame>
  <img src="/images/api-keys-step-1.png" />
</Frame>

Selecting API keys should present this interface to you:

<Frame>
  <img src="/images/api-keys-step-2.png" />
</Frame>

Clicking Generate New API Key will present this form:

<Frame>
  <img src="/images/api-keys-step-3.png" />
</Frame>

Add a description and click Generate. You'll be presented with your API key.

<Warning>This is the only time you'll be shown the key. Please make sure to copy it.</Warning>

<Frame>
  <img src="/images/api-keys-step-4.png" />
</Frame>

Clicking close will take you back to your API key list:

<Frame>
  <img src="/images/api-keys-step-5.png" />
</Frame>
